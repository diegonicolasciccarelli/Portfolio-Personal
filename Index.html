Mis Proyectos

Aplicación de escritorio desarrollada en .Net

Aplicación Winform de escritorio de administración de artículos desarrollada en .Net utilizando Microsoft Visual Studio 2019, con conexión a base de datos utilizando SQL, utilizando la lógica de la programación orientada a objetos, desarrollando la programación en capas para un mejor ordenamiento de la estructura del programa. La misma fue desarrolla en C#.

Programa de consola de gestión de ventas de Restaurant

Programa de consola de administración de ventas de un Restaurant desarrollada en C++ utilizando Codeblocks como compilador. Se realizó utilizando la lógica de la programación orientada a objetos implementando el manejo de archivos .dat para el almacenamiento de datos de las ventas, trabajadores, platos del programa, entre otros, lo cual permite conservar la información luego de cerrarse la consola de la aplicación y conservar todos los cambios realizados en su ejecución.

Juego de consola: La Generala

Juego de consola desarrollado en C++ utilizando Codeblocks como compilador. Se realizó siguiendo las reglas del tan conocido juego de dados , La Generala. El misma tiene un modo de un jugador, dos jugadores, y puntuación mas alta obtenida de los juegos, permitiéndonos registrar los nombres de los jugadores, sus puntajes, tirar los dados y elegir si continuar tirando cada ronda o elegir la cantidad de dados y cuales volver a tirar, obteniendo los puntajes correspondientes a cada combinación. ¡Si sale generala servida ganas el juego!
